# PizzaBase 🍕

A pizza ordering app and admin platform.

## Tooling 

- React Native
- Expo
- Expo-Router
- Supabase

## Features
- User and Admin User Interface
- Food Menu & Cart
- User & Admin Authentication
- Ordering & Order Status
- Backend: Create, Read, Edit and Delete Products.
- Supabase Database
- Stripe Payments
- Notifications 

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
